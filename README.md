# Customer Segmentation

Ini bertujuan untuk melakukan segmentasi pelanggan menggunakan algoritma K-Means Clustering berdasarkan data transaksi dari Superstore.

## Struktur File
* `Data/Superstore.csv` - Dataset utama.
* `customer_segmentation.ipynb` - Jupyter Notebook proses analisis dan modeling.
* `Pipfile` & `Pipfile.lock` - Manajemen *dependencies* menggunakan Pipenv.

## Cara Menjalankan
1. Pastikan kamu sudah menginstal Pipenv.
2. Jalankan perintah `pipenv install` untuk menginstal semua library.
3. Buka notebook `customer_segmentation.ipynb`.
